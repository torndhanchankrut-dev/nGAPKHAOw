# 🚀 Quick Start: Deploy to GitHub Pages

## ⚡ Super Fast Method (5 Minutes)

### Step 1: Run Setup Script
Double-click: `git-setup.bat`

This will:
- ✅ Initialize Git
- ✅ Add all files
- ✅ Create first commit

### Step 2: Create GitHub Repository
1. Go to: https://github.com/new
2. Name: `GAPKHAO`
3. Public repository
4. **Don't** add README
5. Click **Create**

### Step 3: Connect & Push
Copy these commands from GitHub and run in CMD:

```bash
git remote add origin https://github.com/torndhanchankrut-dev/GAPKHAO.git
git branch -M main
git push -u origin main
```

### Step 4: Enable GitHub Pages
1. Go to repository **Settings**
2. Click **Pages** (left sidebar)
3. Source: Branch `main`, Folder `/ (root)`
4. Click **Save**

### Step 5: Done! 🎉
Your site is live at:
```
https://torndhanchankrut-dev.github.io/GAPKHAO/
```

## 🔄 Update Your Site Later
